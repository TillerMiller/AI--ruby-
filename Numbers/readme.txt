Numbers Sample

This archive contains samples of the Numbers Corpus. The Numbers Corpus is a collection of naturally produced numbers. The utterances were taken from other CSLU telephone speech data collections, and include isolated digit strings, continuous digit strings, and ordinal/cardinal numbers. A total of approximately 15,000 files are included in this corpus.

The /docs directory contains the most up-to-date documentation
for the entire corpus.

The /speech directory contains all the wav files separated into many subdirectories
The /trans directory contains the transcription file that correspond to the wav files 
The /labels directory contains the labels for each wav file



Please check our web site http://cslu.cse.ogi.edu/corpora for updates.
